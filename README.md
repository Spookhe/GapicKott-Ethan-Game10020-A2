## This Event Driven System used to showcase interactions through using elemental wands.
## Players must shoot elemental orbs at crystals in order to trigger events.
## Fire + Water = Steam (Creates a geyser that lifts the player upwards)
## Water + Shock = Electrify (Powers and unlocks doors)
## Shock + Fire = Explosion (Destroys objects uncovering new areas)

## Controls: 
###  LMB: Fires Elemental Orb forward
###  WASD: Movement Keys
###  ScrollWheel: Cycles through Elemental wands
